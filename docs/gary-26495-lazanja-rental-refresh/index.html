What's NOT Included

The following are excluded unless added via written change order.

  • Furniture removal, moving, or storage
  • Landscaping or any exterior work beyond windows/door

  • Any damage discovered during demolition (subfloor rot, mold, failed plumbing, live electrical in soffit, etc.) will be assessed on-site and addressed as needed — priced and approved before work proceeds
  • Any work not explicitly listed in this proposal
