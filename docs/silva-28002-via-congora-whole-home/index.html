01 — Project Overview

About This Project

This proposal covers a comprehensive whole-home remodel of the residence at 28002 Via Congora Mission. The scope includes a full kitchen renovation, master and guest bathroom refreshes, new LVP flooring throughout, whole-home ceiling and paint, windows and doors, a stone fireplace surround, and all associated demo, drywall, plumbing, and electrical work.

All work will be performed by Timberline Build Co and our trusted subcontractor team. Justin Sherman will be on-site and personally managing this project from start to finish.
