Phase 1 — Scope of Work

Guest Bathroom

01

Demolition

Remove existing shower, tile, flooring, vanity, and fixtures. Haul off all debris. Prep all surfaces for new construction.

02

Blocking & Framing

Frame pony wall at shower entrance. Block shampoo niche in shower wall. All blocking secured for tile and fixture attachment.

03

Hot Mop & Waterproofing

Hot mop shower pan. Apply full waterproofing membrane throughout shower area before any tile work begins.

04

Tile — Floated Shower & Bath Floor

Hand-float shower walls with concrete mortar bed. Set all tile:

  • Shower floor (client supplies tile)
  • Shower walls — floated concrete bed (client supplies tile)
  • Shower dam (curb) — client-supplied countertop slab material
  • Bathroom floor (client supplies tile)
  • Shampoo niche — client-supplied countertop slab material

Includes all setting materials: thinset, mortar, waterproofing membrane, grout, sealer.

05

Pony Wall Glass

Supply and install glass panel on pony wall at shower entrance. Properly sealed and secured.

06

Vanity & Countertop

Install client-provided vanity. Fabricate and install client-provided countertop with sink cutout. Set client-provided sink.

07

Plumbing

New angle stops behind wall with drywall repair. Shower valve, diverter, and trim out. Connect sink. Set toilet. All drains remain in existing locations — no slab work required.

08

Electrical

2 recessed can lights over vanity, 1 recessed can light over toilet area. Up to 2 outlets. 2 switches. Vent fan switch. All to code.

Client provides vent fan. Labor to install and vent through attic to exterior included.
09

Vent Fan Install

Install client-provided vent fan. Duct vented through attic to exterior.

10

Paint

Full bathroom paint upon completion.

Not Included — Phase 1
  • ✕ All finish materials — tile, vanity, countertop, sink, fixtures, vent fan, toilet (client-supplied)
  • ✕ Hidden damage beyond standard scope — handled as change order with written approval

Phase 2 — Scope of Work

Master Bathroom

01

Demolition

Full demo of existing shower, tile, flooring, vanity, and fixtures. Remove existing window. Haul off all debris. All drywall on the exterior wall removed — water damage repair.

Any additional hidden damage discovered during demo will be documented and handled via change order with written approval before proceeding.
02

Exterior Wall — Drywall & Insulation

Remove drywall on exterior wall. Replace insulation. Install new moisture-resistant drywall. Tape, mud, and texture to match.

If stud or structural damage is found beyond standard drywall removal, it will be assessed and billed as a change order with written approval before proceeding.
03

Framing

  • Frame toilet room walls with open doorway (no door)
  • Frame 2 new window openings (remove old window)
  • Frame shower bench
  • Block shampoo niche in shower wall
  • Frame pony wall on shower entrance side
  • Recess framing for 2 medicine cabinets
04

2 New Windows

Install 2 client-provided windows. Includes proper flashing, waterproofing at rough opening, exterior stucco patch and paint, and interior casing installation.

Client provides windows and interior window casing.
05

Interior Drywall

Hang, tape, mud, and texture all new partition walls (toilet room, all disturbed areas). Moisture-resistant drywall in wet areas. Match existing texture throughout.

06

Hot Mop, Waterproofing & Full Tile Package

Hot mop shower pan. Full waterproofing membrane. Hand-float shower walls with concrete mortar bed. Set all tile:

  • Shower floor (client supplies tile)
  • Shower walls — floated concrete bed (client supplies tile)
  • Full bathroom floor (client supplies tile)
  • Shower bench top — client-supplied countertop slab material
  • Shampoo niche — client-supplied countertop slab material
  • Shower dam (curb) — client-supplied countertop slab material

Includes all setting materials: thinset, mortar, waterproofing membrane, grout, sealer.

07

Countertop Slab Trim — Shower Details

Fabricate and install trim from client-provided countertop slab material on: shower side edges, bench top, shampoo niche surfaces, and shower dam (curb). All cutting and fabrication labor included.

08

Hinged Frameless Glass Shower Door

Supply and install frameless hinged glass shower door. Measured to fit, professionally installed with proper sealing and hardware.

10

Plumbing

New angle stops behind wall with drywall repair. Shower valve, diverter, and trim out. Shower head and wand trim out. Connect 2 sinks. Set toilet. All drain lines remain in existing locations — no slab cutting required.

Toilet relocation: if rough-in requires moving after demo is complete, this will be assessed and priced as a change order at that time.
11

Electrical — Full Package + Dedicated Circuit

Dedicated 20-amp circuit run from panel to master bath. Plus:

  • 2 separate dimmers for medicine cabinets
  • 2 outlets behind medicine cabinets (in-cabinet power)
  • 7 recessed can lights throughout
  • Hardwired towel warmer with countdown timer switch
  • Replace vent fan (client provides fan)

All wiring to code. Panel connection included.

12

Medicine Cabinet Installation (2)

Recess and install 2 client-provided medicine cabinets into framed wall openings. Properly secured, leveled, and finished. Blocking done during framing phase.

Client provides medicine cabinets.
13

Paint

Full master bathroom paint upon completion. Touch-up at all window areas and any disturbed surfaces.
