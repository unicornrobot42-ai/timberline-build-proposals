Scope of Work

01

Demolition

Full demo of existing shower, tile, flooring, vanity, and fixtures throughout the master bath. Remove existing window. Haul off all debris. All drywall on the exterior wall will be removed due to water damage.

Note: Any additional hidden damage discovered during demo will be documented and handled via a change order with your written approval before proceeding.
02

Exterior Wall — Drywall, Insulation & Water Damage Repair

Remove all drywall on the exterior wall. Replace insulation. Install new moisture-resistant drywall. Tape, mud, and texture to match. Properly prepped before any window or tile work begins.

03

Framing

Full framing package for the new layout:

  • Frame toilet room walls with framed opening (no door)
  • Modify existing window opening — remove old window, frame 2 new window openings
  • Frame shower bench
  • Block shampoo niche in shower wall
  • Frame pony wall on shower entrance side
  • Recess framing for 2 medicine cabinets
04

2 New Windows

Install 2 client-provided windows into new framed openings. Includes proper flashing, waterproofing at rough opening, exterior stucco patch and paint, and interior casing installation.

Client provides windows and interior window casing. All framing, flashing, sealing, stucco patch/paint, and casing install included.
05

Interior Drywall

Hang, tape, mud, and texture all new partition walls (toilet room, any disturbed areas). Match existing texture throughout. Moisture-resistant drywall used in wet areas.

06

Hot Mop, Waterproofing & Full Tile Package

Hot mop shower pan, apply full waterproofing membrane throughout shower, then hand-float shower walls with concrete mortar bed. Install all tile:

  • Shower floor tile (client supplies tile)
  • Shower wall tile — floated concrete walls (client supplies tile)
  • Full bathroom floor tile (client supplies tile)

Floated shower walls — hand-floated to a true plane with mortar before tile is set. The correct method for a high-quality, long-lasting shower. Includes all setting materials: thinset, mortar, waterproofing membrane, grout, sealer.

Bench top, shampoo niche, and shower dam will be finished with client-supplied countertop slab material (see item 07 below). Accent wall tile is priced as an optional add-on below.
07

Countertop Slab Trim — Shower Details

Fabricate and install trim from client-provided countertop slab material throughout the shower. Applied to: shower side edges, bench top, shampoo niche surfaces, and shower dam (curb). Creates a continuous stone finish that ties the shower and vanity together. Includes all cutting and fabrication labor.

08

Countertop + 2 Sinks

Fabricate and install client-provided countertop with double sink cutouts. Set both client-provided sinks. All connections and finish work included.

Client provides countertop and sinks. Plumbing connections are a separate optional add-on.
09

Hinged Frameless Glass Shower Door

Supply and install frameless hinged glass shower door. Measured to fit, professionally installed with proper sealing and hardware.

10

Medicine Cabinet Installation (2)

Recess and install 2 client-provided medicine cabinets into framed wall openings. Properly secured, leveled, and finished. Includes blocking during framing phase.

Client provides medicine cabinets. Electrical connections (dimmers + outlets) are a separate optional add-on.
11

Paint

Full master bathroom paint upon completion. Touch-up at all window areas and any surfaces disturbed during construction.
