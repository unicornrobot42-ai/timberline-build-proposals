Project Overview

What We're Building

This proposal covers a full bathroom remodel at 609 E Avenida San Juan, San Clemente. The scope includes complete demolition, construction of a fully enclosed cave-style shower, full tile installation throughout, new electrical and plumbing, vanity and medicine cabinet installation, exhaust fan, door replacement, and ceiling repaint.

All finish materials are provided by the client — including tile, vanity, medicine cabinet, countertop slab, sconce fixtures, and ceiling paint. Timberline supplies all labor, rough materials, and installation expertise.

Scope of Work

What's Included

  • Full Demolition Complete tear-out of all existing finishes, fixtures, and flooring. Removal of partition wall between vanity and toilet area.
  • Shower Wall Framing Frame and build fully enclosed shower walls — cave-style design, no glass enclosure required.
  • Drywall & Wall Prep Patch and repair all non-wet areas after demo. Hang new drywall as needed.
  • Cement Board + RedGuard Waterproofing Install cement board and full RedGuard membrane in all wet zones (shower walls, shower floor area).
  • Hot Mop — Shower Pan Professional hot mop of shower pan for complete waterproof integrity prior to tile installation.
  • Tile Installation — All Areas (Labor Only) Set, grout, and finish tile on all bathroom walls and floor. Client supplies all tile.
  • Plumbing — Shower Valve Installation Supply and set shower valve with proper water connections.
  • Plumbing — Toilet Drain Relocation Relocate toilet drain to accommodate new layout.
  • Plumbing — Set Toilet & Vanity Reconnect Set toilet in new location, reconnect all vanity plumbing.
  • Electrical — 4 Recessed Can Lights Supply and install four recessed LED cans with proper electrical runs and switches.
  • Electrical — 2 Sconces (Wire & Install) Run wire and install two vanity sconces. Client supplies fixtures.
  • Electrical — Exhaust Vent Fan Supply and install new exhaust fan with exterior venting. No existing fan in this bathroom.
  • Vanity Installation (Labor Only) Install client-supplied vanity, make plumbing connections.
  • Medicine Cabinet — Frame & Install (Labor Only) Frame recessed opening and install client-supplied medicine cabinet over sink.
  • Ceiling Repaint (Labor Only) Prep and apply finish coat to bathroom ceiling. Client supplies paint.
  • Door & Casing Replacement — 28" RH Install 6-panel door with generic casing. Includes door, hardware, and casing installation.
