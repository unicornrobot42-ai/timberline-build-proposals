Overview
Project Overview

This proposal covers a full bathroom transformation at 260 Wicker, Irvine — including a tub-to-shower conversion with a custom tiled 32×60 walk-in shower, new bathroom floor tile throughout, and all associated plumbing, glass, and finish work. Optional add-ons for vanity installation and a lit mirror are detailed separately.

Our goal is a clean, modern bathroom that's completed on time and built to last — with materials staged and ready before we start so your timeline stays on track.

Concept render — final appearance subject to material selections

What's Included
Scope of Work
  • Full demo — remove existing tub, tile, and fixtures
  • Remove and reset toilet
  • Shower blocking — framing for 32×60×107 shower enclosure and niche
  • Hot mop shower pan (waterproof membrane)
  • Tile labor — shower walls, built-in niche, and shower floor
  • Tile labor — bathroom floor
  • Plumbing — install new shower valve, trim, and fixtures
  • Frameless sliding glass enclosure (32×60)
  • Drywall and moisture-resistant backer as needed

  • Install client-provided medicine cabinet
