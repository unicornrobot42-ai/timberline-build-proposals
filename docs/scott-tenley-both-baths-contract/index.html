Scope of Work

Timberline Build Co agrees to furnish all labor, supervision, and contractor-supplied materials as described below for the remodel of two bathrooms at the property address. All finish materials (tile, vanity, fixtures, glass, wallpaper, hardware, and accessories) are excluded from this contract. Client is responsible for purchasing or providing all finishes prior to the scheduled start of each phase.

Line Item Description Notes
Downstairs Guest Bathroom — Tub-to-Shower Conversion
Demo Remove existing tub, surround, and prep shower area Labor only
Hot Mop Waterproof shower pan installation Required for walk-in shower
Tile Labor Full shower tile installation — walls and floor Client provides tile
Plumbing Tub-to-shower conversion, new rough-in, shower valve and trim installation Client provides fixtures
Vanity Install Install client-supplied vanity, sink, and faucet Client provides vanity + fixtures
Electrical 3 new recessed can lights, replace vent fan, relocate vanity electrical from overhead to 2 wall sconces Client provides sconce fixtures and vent fan
Wainscoting Install wood wainscoting to approximately 42" height around perimeter Labor + framing materials included
Wallpaper Labor Hang client-supplied wallpaper on all drywall surfaces above wainscoting (excluding ceiling) Client provides wallpaper
Shampoo Niche Frame and tile one recessed shampoo niche in shower Client provides tile
Glass Install Install client-supplied single panel glass with opening opposite shower fixture Client provides glass panel
Paint Paint entire bathroom including wainscoting — labor and standard paint included
Upstairs Bathroom — Restore to Working Order
Tile Labor Install client-supplied tile flooring and baseboard tile Client provides tile
Drywall Repair Patch and repair drywall as needed
Paint Paint entire bathroom — labor and standard paint included
Reset Toilet Remove, reset, and reseal existing toilet

What Is NOT Included

  • All finish materials — tile, grout, vanity, sink, faucets, shower fixtures, glass panel, wallpaper, sconces, vent fan, hardware, accessories, and mirrors
  • Building permits (excluded from this contract — client may request separately)
  • Structural modifications beyond scope listed above
  • Flooring in downstairs bathroom (existing flooring to remain)
  • Appliance or furniture moving not related to scope
  • Any work discovered behind walls not identified prior to demo (subject to change order)
Client Responsibility — Finishes Client agrees to purchase and have all finish materials on-site and approved prior to the scheduled start of each phase. Delays caused by late or missing materials may result in schedule adjustments and/or additional costs billed at time-and-materials rates.
