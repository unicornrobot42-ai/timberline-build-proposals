01 — Project Overview
1

A Fresh Kitchen,
Same Great Footprint

This project transforms the existing kitchen at the Montrose residence with a complete cabinet replacement, new quartz countertops, updated sink and faucet, and under-cabinet lighting — all within the current floor plan footprint. The result: a modern, functional kitchen with clean lines and durable finishes that will serve the home for decades.

Concept render — finishes subject to client selection

02 — Scope of Work
2

What's Included

Demo: Remove all existing upper cabinets, base cabinets, pantry unit, island cabinets, countertops, backsplash tile, and sink

Cabinet Install (Labor): Install all new cabinets — base, uppers, corner unit, tall pantry, and island. Client supplies cabinets.

Quartz Countertops: Template, fabricate, and install quartz countertops on all perimeter runs and island

Plumbing: Disconnect and reconnect sink plumbing — supply lines, drain, and faucet installation

Appliance Install: Reconnect range, dishwasher, and refrigerator to existing connections

Vent Hood Install: Mount and connect new vent hood to existing ductwork

Under-Cabinet Lighting: Install LED under-cabinet lighting throughout all upper cabinet runs

Drywall Patching: Patch and repair all wall surfaces exposed during cabinet demo

Paint Touch-Up: Touch up walls after cabinet installation is complete

Debris Removal: Full haul-off of all demo material and job-site debris

03 — Exclusions
3

What's NOT Included

  • Cabinet material cost (client-supplied)
  • Backsplash tile supply and installation (available as add-on)
  • New appliance purchase
  • Electrical panel upgrades or new circuit runs
  • Flooring repair or replacement
  • Window or door modifications
  • Permits (available as add-on — typically $500–$1,500 in OC)
  • Any work outside the kitchen footprint
