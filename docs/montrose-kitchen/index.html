Scope of Work

Complete Scope Breakdown

All work performed by licensed Timberline crew. Cabinet cost excluded — client supplies.

1

Demo & Site Prep

Day 1–3

Full removal of all existing cabinetry — uppers, lowers, corner unit, tall pantry, and island base cabinets. Existing granite countertops, full mosaic tile backsplash, and sink disconnected and removed. Debris hauled off-site. Wall surfaces assessed and prepped for patching.

  • Remove all upper cabinets
  • Remove all base cabinets
  • Remove corner unit + tall pantry
  • Remove island base cabinets
  • Remove all countertops (granite)
  • Remove full mosaic backsplash tile
  • Disconnect + remove sink
  • Full debris haul-off
2

Cabinet Installation

Labor Only — Client Supplies Cabinets

Full installation of all client-supplied cabinets. This is a complete cabinet set — base runs, all uppers, corner cabinet (lazy susan or blind corner), tall floor-to-ceiling pantry unit, and island base cabinets. Installation includes shimming, leveling, scribing to walls, and securing per manufacturer specifications. All cabinets are set plumb, level, and square.

  • All base cabinets set + leveled
  • All upper cabinets hung
  • Corner cabinet (lazy susan / blind)
  • Tall pantry unit install
  • Island base cabinet set
  • Scribing + wall fitting
  • Hardware holes pre-drilled
  • Door + drawer adjustment
Cabinet Cost (Not Included)

Cabinet materials are client-supplied. Budget $10,000–$18,000 for semi-custom cabinets at ~28 LF. We coordinate delivery timing to align with installation.

3

Quartz Countertops

Perimeter Runs + Island · Fabrication + Install

Timberline coordinates template, fabrication, and installation of quartz countertops across all perimeter cabinet runs and the island. Countertop vendor templates after cabinets are fully set. Typical lead time is 1–2 weeks from template to install. Sink cutout is included in fabrication.

  • Template after cabinet set
  • Full perimeter quartz counters
  • Island quartz countertop
  • Sink cutout included
  • Edge profile of your choice
  • Professional installation
4

Plumbing, Sink & Appliances

Reconnect + Hook-Up

All plumbing reconnections, sink install, faucet hook-up, and appliance reconnections. This includes dishwasher, range, and refrigerator reconnect to existing supply/drain connections. Vent hood is mounted and connected to existing ductwork.

  • Sink install (undermount)
  • Faucet hook-up
  • Supply lines + drain connect
  • Dishwasher reconnect
  • Range reconnect
  • Refrigerator reconnect
  • Vent hood mount + duct connect
  • Leak test + final check
5

Finish Work — Lighting, Drywall & Paint

Final Phase

Under-cabinet LED lighting installed throughout all upper cabinet runs. Post-demo drywall patching and skim coat where needed. Final paint touch-up on all walls after installation is complete. Site cleanup and final walkthrough.

  • Backsplash tile installation (labor)
  • Under-cabinet LED lighting install
  • Drywall patch (post-demo)
  • Skim coat where needed
  • Wall paint touch-up
  • Final site cleanup
  • Punch list walkthrough
