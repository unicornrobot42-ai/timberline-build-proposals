What's Included

Scope of Work

01

Demolition

Complete demo of existing shower, tile, and fixtures. All debris removed and hauled off site.

Note: Any hidden damage discovered during demo (soft substrate, rotted framing, mold) will be documented and handled via a change order with your written approval before proceeding.
02

Drywall — Exterior Wall

Remove and replace all drywall on the exterior wall. Properly prep for window opening and ensure a clean, flat surface for finish work.

03

New Construction Window

Frame opening, install client-provided window, seal interior and exterior. Repair and paint stucco on exterior. Install client-provided window casing on interior.

Client provides window and window casing. Labor, framing, sealing, and stucco patch/paint included.
04

Block Shower + New Dam / Curb

Block shower to new dimensions and build new shower curb/dam. Properly formed and secured for tile installation.

05

Hot Mop, Waterproofing & Full Tile Package

Hot mop shower pan, apply full waterproofing membrane, hand-float shower walls with concrete mortar bed, then install all tile throughout:

  • Shower floor tile (client supplies tile)
  • Shower wall tile — floated concrete walls, no accent wall (client supplies tile)
  • Bathroom floor tile (client supplies tile)

Floated shower walls means walls are hand-floated to a true plane with mortar before tile is set — the correct method for a high-quality, long-lasting shower. Includes all setting materials — thinset, mortar, waterproofing membrane, grout, sealer.

No accent wall in this scope. Accent wall available as a change order if desired.
06

Countertop & Sink Installation

Fabricate and install client-provided pre-fab countertop. Set client-provided sink. All connections and finish work included.

07

Frameless Glass Euro Slider Shower Door

Supply and install frameless glass euro slider shower door. Measured to fit, professionally installed with proper sealing and hardware.

08

Paint

Full bathroom paint upon completion. Touch-up paint at window area and any areas disturbed during construction.
