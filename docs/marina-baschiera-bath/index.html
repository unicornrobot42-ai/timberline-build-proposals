Project Overview
What We're Building

Complete guest bathroom remodel at 2348 Littleton Circle, Costa Mesa. Tub-to-shower conversion with linear drain, frameless glass enclosure, recessed niche, new vanity and countertop installation, updated ventilation with ducting, drywall repair, and fresh paint.

All finish materials are provided by the client — tile, vanity, countertop, and fixtures. Timberline supplies all labor, rough materials, and installation expertise.

Scope of Work
What's Included
  • Complete DemolitionRemove existing tub, surround, and all existing fixtures. Prep space for shower conversion.
  • Hot Mop WaterproofingProfessional hot mop of shower pan for complete waterproof integrity.
  • Linear Drain InstallationInstall linear drain with proper floor slope for modern, curbless-look shower.
  • Tile Installation — All Areas (Labor Only)Set, grout, and finish tile on shower walls, floor, and niche. Client supplies all tile.
  • Recessed Shower NicheFrame and build recessed niche for shampoo storage. Tile billed under tile labor.
  • Frameless Glass Shower EnclosureInstall frameless glass panel or door. Configuration confirmed at site visit.
  • Vanity Installation (Labor Only)Remove old vanity, install client-supplied vanity, make all plumbing connections.
