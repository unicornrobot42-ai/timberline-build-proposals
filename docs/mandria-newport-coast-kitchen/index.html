Project Overview

Full Kitchen Remodel — Exact Footprint + Living Room Extension

A complete renovation of the existing kitchen within its current footprint, upgraded with new cabinetry, countertops, appliances, plumbing, lighting, and backsplash — plus an extension of cabinetry and a new bar top into the adjacent living room. This scope preserves the existing layout while dramatically updating every visible and functional surface.

An optional wall removal is available to fully open the kitchen to the dining area and is detailed in the Add-Ons section below.

Scope of Work

What's Included

  • Demo — existing cabinets, countertops, flooring, fixtures, and hood; full haul-away
  • Cabinet installation — full kitchen (uppers + lowers + second wall cabinet row), dining peninsula with 12" bar overhang, and living room cabinet extension including bar top
  • Countertops — kitchen, dining peninsula, and living room bar top; material + install (up to $7,000 budget; see Material Allowances)
  • Backsplash installation — labor; client supplies tile
  • Plumbing — sink rough-in and hook-up, refrigerator ice/water line, garbage disposal installation
  • Appliance installation — range, dishwasher, microwave, and refrigerator; labor only
  • Under-cabinet lighting — wiring and installation; client selects fixtures
  • Paint — repair areas only (patched surfaces post-demo)
  • All debris removal and final broom clean included
All pricing assumes existing electrical, plumbing, and structural conditions are to current code. Any upgrades required by inspection will be scoped and billed via change order prior to work proceeding.
Exclusions

What's NOT Included

  • Cabinets — client purchases and delivers all cabinets; contractor installs only
  • Appliances — client purchases and delivers all appliances; contractor installs only
  • Backsplash tile — client selects and purchases; contractor installs
  • Under-cabinet light fixtures — client selects and purchases; contractor installs wiring
  • Whole-home paint — paint scope is repair/patch areas only
  • Wall removal (see Optional Add-On below — priced separately)
  • Flooring — covered under separate whole-home flooring scope
  • Permit fees — client pays city/county directly
Material Allowances

Client-Provided Materials

The following items have allowances built into the investment below. If client-selected materials exceed the stated allowance, the difference is billed as a change order.

  • Countertops (kitchen + peninsula + bar top) — $7,000 allowance (material + fabrication + install)
  • Cabinets — client-supplied; no material allowance required
  • Appliances — client-supplied; no material allowance required
  • Backsplash tile — client-supplied; no material allowance required
Optional Add-On

Available Upgrades

Optional + $3,000
Kitchen Wall Removal — Open to Dining Area
Remove the existing non-load-bearing wall between the kitchen and dining area to create a fully open floor plan. Includes all framing work, drywall repair at ceiling, floor line, and end walls, plus paint on all repaired surfaces.
This wall has been identified as non-load-bearing and non-structural. No engineering, structural drawings, or beam work is required or included in this price. If an on-site inspection reveals unexpected structural conditions, a change order will be issued before any work proceeds.
